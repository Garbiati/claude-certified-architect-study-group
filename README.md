# Grupo de Estudos – Claude Certified Architect (Anthropic)

> **⚠️ Aviso:** Este projeto **não é afiliado à Anthropic**. É uma iniciativa comunitária criada para ajudar engenheiros e arquitetos a se prepararem para a certificação **Claude Certified Architect – Foundations**.

---

## 📖 Visão Geral do Projeto

Este repositório é um **hub de estudos colaborativo** para a certificação Claude Certified Architect da Anthropic. O conteúdo é compilado a partir de múltiplas fontes públicas, traduzido e adaptado para o **Português (pt-BR)**, e enriquecido com os aprendizados práticos da nossa comunidade.

O repositório evolui continuamente com as contribuições da comunidade: novos materiais, experimentos, notas de sessão e traduções são adicionados regularmente. Contribuições em inglês também são bem-vindas.

---

## 🎯 Objetivo do Grupo de Estudos

Criar um espaço colaborativo onde engenheiros e arquitetos possam:

- 📚 Organizar materiais de estudo para a certificação
- 🌐 Compilar recursos públicos relevantes
- 🇧🇷 Traduzir e adaptar conteúdo para o Português (pt-BR)
- 🔬 Documentar experimentos e aprendizados práticos
- 🤝 Apoiar uns aos outros durante a preparação para a prova

---

## 📅 Formato do Grupo de Estudos

As sessões de estudo acontecem **duas vezes por semana**, às **terças e quintas-feiras**.

Durante as sessões, o grupo irá:

- Revisar os tópicos da certificação
- Discutir conceitos de arquitetura de IA
- Testar prompts e agentes
- Compartilhar experimentos práticos
- Atualizar o repositório com os aprendizados

---

## 💬 Comunidade

A comunicação principal acontece via **WhatsApp** e **Discord**.

Caso tenha interesse em participar, **comente no post do LinkedIn** e o link de convite será compartilhado com você.
