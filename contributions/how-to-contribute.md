# Como Contribuir

Obrigado por querer contribuir com o grupo de estudos **Claude Certified Architect – Foundations**! 🙌

Este repositório é construído colaborativamente. Toda contribuição — desde a correção de um erro de digitação até a adição de novos materiais — é muito bem-vinda.

---

## 🌱 Tipos de Contribuição

Você pode contribuir de diversas formas:

- **Adicionar recursos:** artigos, vídeos, cursos, ferramentas úteis
- **Corrigir erros:** ortografia, informações incorretas, links quebrados
- **Melhorar materiais:** expandir explicações, adicionar exemplos, melhorar formatação
- **Traduzir conteúdo:** adaptar materiais para português (pt-BR) ou outros idiomas
- **Documentar sessões:** adicionar notas e aprendizados das sessões de estudo
- **Propor melhorias:** sugerir novas seções, tópicos ou formatos

---

## 🛠️ Como Contribuir via GitHub

### 1. Fork do repositório

Faça um fork deste repositório para a sua conta GitHub.

```bash
# Clone o seu fork localmente
git clone https://github.com/SEU_USUARIO/claude-certified-architect-study-group.git
cd claude-certified-architect-study-group
```

### 2. Crie uma branch para sua contribuição

```bash
git checkout -b minha-contribuicao
```

Use um nome descritivo para a branch, como:
- `add-rag-resources`
- `fix-study-plan-week2`
- `translate-exam-topics-en`

### 3. Faça suas alterações

Edite ou adicione os arquivos necessários. Siga as diretrizes de estilo abaixo.

### 4. Commit e push

```bash
git add .
git commit -m "feat: adiciona recursos sobre RAG na semana 3"
git push origin minha-contribuicao
```

### 5. Abra um Pull Request

Abra um Pull Request (PR) para a branch `main` deste repositório. Descreva claramente:
- O que foi alterado
- Por que a mudança é útil
- Qualquer contexto adicional relevante

---

## 📝 Diretrizes de Estilo

- **Idioma principal:** Português (pt-BR). Contribuições em inglês são aceitas.
- **Formatação:** Use Markdown padrão.
- **Linguagem:** Clara, técnica e acessível.
- **Fontes:** Cite sempre a origem dos materiais adicionados.
- **Estrutura:** Mantenha a estrutura existente de pastas e arquivos.

---

## 📁 Estrutura do Repositório

Ao adicionar novos materiais, use as pastas corretas:

| Pasta | Conteúdo |
|---|---|
| `/docs` | Plano de estudos, tópicos do exame, recursos |
| `/sessions` | Notas e materiais das sessões de estudo |
| `/community` | Informações sobre como participar da comunidade |
| `/translations` | Versões traduzidas dos documentos principais |

---

## 📋 Contribuições Sem Código (Issues)

Não sabe usar Git? Sem problema! Você ainda pode contribuir:

1. Abra uma **Issue** neste repositório.
2. Descreva sua sugestão, correção ou recurso a ser adicionado.
3. Um membro do grupo irá implementar ou te ajudar a fazer isso.

---

## ✅ Checklist para Pull Requests

Antes de abrir um PR, verifique:

- [ ] O conteúdo é relevante para a certificação *Claude Certified Architect – Foundations*
- [ ] O texto está em português (pt-BR) ou inglês
- [ ] Links e referências foram verificados e funcionam
- [ ] A formatação Markdown está correta
- [ ] Não há conteúdo proprietário ou protegido por direitos autorais
