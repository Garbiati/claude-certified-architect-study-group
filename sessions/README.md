# Sessões de Estudo

Este diretório documenta as sessões de estudo do grupo **Claude Certified Architect – Foundations**.

---

## 📋 Sobre as Sessões

As sessões acontecem **duas vezes por semana**:

- **Terças-feiras** — revisão de conteúdo teórico e tópicos da certificação
- **Quintas-feiras** — sessão prática: testes de prompts, agentes e experimentos

---

## 🗂️ Estrutura de uma Sessão

Cada sessão é documentada em um arquivo Markdown neste diretório, com o formato:

```
YYYY-MM-DD-tema-da-sessao.md
```

**Exemplo:** `2025-04-01-fundamentos-llm.md`

### Conteúdo de uma nota de sessão

```markdown
# Sessão — [Tema da Sessão]
**Data:** DD/MM/AAAA
**Participantes:** N pessoas

## Tópicos Discutidos
- ...

## Experimentos Realizados
- ...

## Aprendizados e Insights
- ...

## Recursos Compartilhados
- ...

## Próximos Passos
- ...
```

---

## 📝 Como Documentar uma Sessão

1. Crie um arquivo com o formato `YYYY-MM-DD-tema.md` nesta pasta.
2. Use o template acima como base.
3. Envie via Pull Request ou compartilhe nos canais da comunidade para que alguém do grupo faça o commit.

Consulte [`/contributions/how-to-contribute.md`](../contributions/how-to-contribute.md) para mais detalhes sobre como contribuir.
