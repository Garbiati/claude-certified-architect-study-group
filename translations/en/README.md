# English Translations

This folder contains English translations of the main repository content.

## Available Translations

*Translations are coming soon. Contributions are welcome!*

## How to Contribute a Translation

1. Create a new Markdown file in this folder mirroring the structure of the source file.
2. Use the naming convention that matches the original file (e.g., `study-plan.md`, `exam-topics.md`).
3. Open a pull request with your translation.

See [how to contribute](../../contributions/how-to-contribute.md) for more details.
