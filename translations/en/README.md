## How to Contribute a Translation

1. Copy the original Portuguese document to this folder.
2. Translate the content to English, preserving the Markdown structure.
3. Submit a Pull Request with your translation.

See [how-to-contribute.md](../../contributions/how-to-contribute.md) for more details.
