# Traducciones al Español

Esta carpeta contiene traducciones al español del contenido principal del repositorio.

## Traducciones Disponibles

*Las traducciones están en camino. ¡Las contribuciones son bienvenidas!*

## Cómo Contribuir con una Traducción

1. Crea un nuevo archivo Markdown en esta carpeta que refleje la estructura del archivo fuente.
2. Usa la misma convención de nombres del archivo original (p. ej., `study-plan.md`, `exam-topics.md`).
3. Abre un pull request con tu traducción.

Consulta [cómo contribuir](../../contributions/how-to-contribute.md) para más detalles.
