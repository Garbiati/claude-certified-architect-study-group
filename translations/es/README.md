## Cómo Contribuir una Traducción

1. Copia el documento original en portugués a esta carpeta.
2. Traduce el contenido al español, preservando la estructura Markdown.
3. Envía un Pull Request con tu traducción.

Consulta [how-to-contribute.md](../../contributions/how-to-contribute.md) para más detalles.
